# 🚨 CRITICAL: FILE NAMING RULE

**Updated:** March 3, 2026  
**Applies to:** ALL team members (Holy Spirit + Deciples 1-12)

---

## ⚠️ MANDATORY RULE

**Every file you create MUST use your name/number as a prefix.**

This prevents everyone from accidentally overwriting each other's work!

---

## ✅ CORRECT EXAMPLES

### If you are Deciple 1:
```
deciple-1-diary.md        ✅
deciple-1-notes.md        ✅
deciple-1-tasks.md        ✅
deciple-1-meeting-notes.md ✅
```

### If you are Deciple 5:
```
deciple-5-diary.md        ✅
deciple-5-notes.md        ✅
deciple-5-ideas.md        ✅
deciple-5-resources.md    ✅
```

### If you are Holy Spirit:
```
holy-spirit-diary.md      ✅
holy-spirit-coordination.md ✅
holy-spirit-updates.md    ✅
```

---

## ❌ WRONG - NEVER DO THIS

```
diary.md                  ❌ (generic - will overwrite!)
notes.md                  ❌ (generic - will overwrite!)
README.md                 ❌ (system file - DON'T TOUCH)
messages.md               ❌ (shared file - DON'T RENAME)
update.md                 ❌ (generic - will overwrite!)
```

---

## 📁 WHERE TO PUT FILES

### Your Personal Folder:
Put files in YOUR folder only:
- Deciple 1 → `deciple-1/`
- Deciple 2 → `deciple-2/`
- ...
- Deciple 12 → `deciple-12/`
- Holy Spirit → `holy-spirit/`

### Shared Resources:
If putting in `shared-resources/`, STILL use your prefix:
```
shared-resources/
  ├── deciple-1-design-mockup.png     ✅
  ├── deciple-3-api-keys.env          ✅
  ├── deciple-7-research-links.md     ✅
  └── holy-spirit-team-directory.md   ✅
```

---

## 🔍 HOW TO CHECK

Before saving, ask yourself:
1. **Does my filename start with my deciple number?**
2. **Is it clear this is MY file?**
3. **Will someone else accidentally save over this?**

If NO to any question → Rename it!
